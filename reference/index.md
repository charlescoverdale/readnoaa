# Package index

## Weather Data

Fetch weather and climate observations

- [`noaa_daily()`](https://charlescoverdale.github.io/readnoaa/reference/noaa_daily.md)
  : Daily weather observations
- [`noaa_monthly()`](https://charlescoverdale.github.io/readnoaa/reference/noaa_monthly.md)
  : Monthly weather summaries
- [`noaa_annual()`](https://charlescoverdale.github.io/readnoaa/reference/noaa_annual.md)
  : Annual weather summaries
- [`noaa_normals()`](https://charlescoverdale.github.io/readnoaa/reference/noaa_normals.md)
  : Climate normals (1991-2020)

## Station Discovery

Find weather stations by location or name

- [`noaa_stations()`](https://charlescoverdale.github.io/readnoaa/reference/noaa_stations.md)
  : Search for weather stations
- [`noaa_nearby()`](https://charlescoverdale.github.io/readnoaa/reference/noaa_nearby.md)
  : Find stations near a location

## Data Access

Generic data access and discovery

- [`noaa_get()`](https://charlescoverdale.github.io/readnoaa/reference/noaa_get.md)
  : Fetch any NCEI dataset
- [`list_datasets()`](https://charlescoverdale.github.io/readnoaa/reference/list_datasets.md)
  : List common NCEI datasets
- [`list_datatypes()`](https://charlescoverdale.github.io/readnoaa/reference/list_datatypes.md)
  : List available data types for a dataset
- [`clear_cache()`](https://charlescoverdale.github.io/readnoaa/reference/clear_cache.md)
  : Clear the readnoaa cache
